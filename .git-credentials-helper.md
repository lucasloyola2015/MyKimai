# Guía para Configurar Git con Token de GitHub

## ⚠️ IMPORTANTE: NO compartas tu token en el chat

Para hacer push de forma segura, sigue estos pasos:

## Opción 1: Usar el token en la URL (temporal)

Cuando hagas push, usa el token directamente en la URL:

```bash
git remote add origin https://TU_TOKEN@github.com/lucasloyola2015/MyKimai.git
```

O para un push único:

```bash
git push https://TU_TOKEN@github.com/lucasloyola2015/MyKimai.git main
```

## Opción 2: Usar Git Credential Manager (Recomendado)

1. Configura el remote sin el token:
```bash
git remote add origin https://github.com/lucasloyola2015/MyKimai.git
```

2. Cuando hagas push, Git te pedirá credenciales:
   - Username: `lucasloyola2015`
   - Password: `TU_TOKEN_AQUI`

3. Git guardará las credenciales de forma segura.

## Opción 3: Usar SSH (Más Seguro)

1. Genera una SSH key:
```bash
ssh-keygen -t ed25519 -C "tu_email@example.com"
```

2. Agrega la clave pública a GitHub:
   - Settings → SSH and GPG keys → New SSH key

3. Cambia el remote a SSH:
```bash
git remote set-url origin git@github.com:lucasloyola2015/MyKimai.git
```
